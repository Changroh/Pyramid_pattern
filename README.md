# Pyramid Pattern in Go

This project prints a pyramid pattern of stars (`*`) based on the number of rows entered by the user.

## How to Run

1. Make sure you have [Go](https://golang.org/dl/) installed.
2. Clone or download this repository.
3. Open a terminal and navigate to the project directory.
4. Run the following command:

   go run main.go

## Usage Example

Enter number of rows: 5

Pyramid pattern for 5 rows:

    *
   **
  ***
 ****
*****

## Requirements

- Go 1.x or later

## Customization

- Try changing the pattern to print numbers or other symbols.
- Add command-line arguments for input instead of interactive prompt.
- Implement inverted or diamond patterns.
